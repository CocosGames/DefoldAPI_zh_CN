粒子特效 API 文档

Version: stable

函数
particlefx.play() 开始播放粒子特效
particlefx.stop() 停止播放粒子特效
particlefx.set_constant() 为粒子特效组件设置着色器常量
particlefx.reset_constant() 为粒子特效组件重置着色器常量
常量
particlefx.EMITTER_STATE_SLEEPING 待机状态
particlefx.EMITTER_STATE_PRESPAWN 重置状态
particlefx.EMITTER_STATE_SPAWNING 加载状态
particlefx.EMITTER_STATE_POSTSPAWN 加载完成状态

Functions

particlefx.play()

particlefx.play(url,[emitter_state_function])

开始播放粒子特效. 这样启动的粒子特效需要使用 particlefx.stop() 手动停止. 其中需要启动的粒子特效用唯一id的URL指定. 即使粒子特效组件所在的游戏对象已被删除, 粒子特效仍会继续播放. 可以使用 particlefx.stop() 停止其播放.

参数

url 需要播放的粒子特效地址.
[emitter_state_function] 可选参数, 当粒子特效状态变化时的回调函数.
self
object 当前对象
id
hash 粒子特效id
emitter
hash 发射器id
state
constant 发射器状态:
  • particlefx.EMITTER_STATE_SLEEPING
  • particlefx.EMITTER_STATE_PRESPAWN
  • particlefx.EMITTER_STATE_SPAWNING
  • particlefx.EMITTER_STATE_POSTSPAWN

实例

游戏对象建立时播放粒子特效. 回调函数接收例子特效路径hash, 发射器id的hash, 以及发射器状态 particlefx.EMITTER_STATE_. 
local function emitter_state_change(self, id, emitter, state)
  if emitter == hash("exhaust") and state == particlefx.EMITTER_STATE_POSTSPAWN then
    -- 播放粒子特效...
  end
end

function init(self)
    particlefx.play("#particlefx", emitter_state_change)
end

particlefx.stop()

particlefx.stop(url)

停止正在播放的粒子特效. 并不删除已存在的粒子. 路径URL决定了是哪个粒子特效需要停止播放.

参数

url 需要关闭的粒子特效url

实例

游戏对象被删除后如何停止其下的粒子特效: 
function final(self)
    particlefx.stop("#particlefx")
end

particlefx.set_constant()

particlefx.set_constant(url,emitter,constant,value)

为粒子特效组件设置一个着色器常量. 这个常量必须先在组件的材质配置里定义好. 用这个方法设置的常量值会自动覆盖材质里定义的默认值. 使用 particlefx.reset_constant 可以恢复默认常量值. 要设置常量的粒子特效组件使用URL指定.

参数

url 需要设置常量的粒子特效的url地址
emitter 粒子发射器id
constant 需要设置的常量名
value 需要设置的常量值

实例

假设有个粒子特效id 叫做 "particlefx", 它 包含一个id 叫做 "emitter" 的发射器且使用默认材质, 材质里有预定义常量 "tint". 对于sprite的自定义材质, 可以使用同样方法设置常量. 把发射的粒子染成红色: 
function init(self)
    particlefx.set_constant("#particlefx", "emitter", "tint", vmath.vector4(1, 0, 0, 1))
end

particlefx.reset_constant()

particlefx.reset_constant(url,emitter,constant)

重置粒子特效组件发射器着色器常量. 这个常量必须先在组件的材质配置里定义好. 用这个方法常量值会重置为材质里定义的默认值. 要设置常量的粒子特效组件使用URL指定.

参数

url 需要重置常量的粒子特效url地址
emitter 粒子发射器id
constant 需要重置的常量名

实例

假设有个粒子特效id 叫做 "particlefx", 它 包含一个id 叫做 "emitter" 的发射器且使用默认材质, 材质里有预定义常量 "tint". 对于sprite的自定义材质, 可以使用同样方法重置常量. 把发射的粒子染成红色: 
function init(self)
    particlefx.reset_constant("#particlefx", "emitter", "tint")
end

常量

particlefx.EMITTER_STATE_SLEEPING

待机状态

此状态下没有粒子产生, 也没有已发射的粒子显示.

particlefx.EMITTER_STATE_PRESPAWN

预备状态

此状态下发射器已被激活但是还未产生粒子. 一般这个状态保持时间很短, 具体要看粒子发射器配置中的延迟设置.

particlefx.EMITTER_STATE_SPAWNING

发射状态

此状态下发射器正在发射粒子.

particlefx.EMITTER_STATE_POSTSPAWN

完成状态

此状态下发射器不再发射粒子, 但是可能有已发射的粒子显示.
